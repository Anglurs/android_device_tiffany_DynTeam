TWRP device configuration for Xiaomi Mi 5X
==============
To build:

```sh
make clean
. build/envsetup.sh
lunch omni_tiffany-userdebug
make recoveryimage -j16

The Xiaomi Mi 5X (codenamed _"Gemini"_) is a high-end smartphone from Xiaomi.

It was announced in February 2018, in three flavours. Release date was April 2018.

| CPU          | Dual-core 2.0GHz Kryo				 |
| GPU          | 510MHz Adreno 506                               |
| Memory       | 4 GB RAM (LPDDR3 933 MHz dual-channel)          |
| Shipped Android version | 7.1.2 with MIUI 9                    |
| Storage      | 64 GB (EMMC 5.1 Flash)                          |
